## What's new

- **New toolbar/app icon — a gray tabby cat.** The cat is now silver-gray with
  tabby markings (forehead "M" stripes, inner-ear shading, a cheek stripe), dark
  eyes, and a pink nose, on the royal-blue tile, framed by the white
  selection-marquee corner brackets (the "snip" motif). The icon is
  **size-adaptive**: the crop brackets, whiskers, and tabby stripes render at
  32/48/128px, while the 16px toolbar icon falls back to a bold, edge-to-edge
  gray cat face so it stays crisp at toolbar size.

That's the whole change — everything else is identical to v1.0.0.

## Chrome Web Store update

1. **Build the package:**
   ```bash
   ./build.sh            # -> dist/catsnip-v1.0.1.zip  (OCR included if lib/tesseract/ present)
   ```
   Sanity-check `unzip -l`: `manifest.json` at the root (version 1.0.1), the
   refreshed `icons/*.png`, and `lib/tesseract/*` present.
2. **Upload `dist/catsnip-v1.0.1.zip`** as a new package version in the
   [Developer Dashboard](https://chrome.google.com/webstore/devconsole).
3. **No other changes needed** — permission justifications, single-purpose
   statement, data-usage answers (no data collected), remote-code answer (none),
   and the privacy-policy URL are all unchanged from v1.0.0. Only the store icon
   (`icons/icon128.png`) is refreshed.
4. **Submit for review.**
